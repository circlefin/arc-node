# arc-node-execution

This crate name is reserved for the Arc node project.

This published placeholder is not a functional library. The real
`arc-node-execution` implementation is currently distributed from
https://github.com/circlefin/arc-node and is kept out of crates.io until the
Reth SDK dependencies Arc uses are available from crates.io at compatible
versions.

Do not depend on version `0.0.0-placeholder` for runtime functionality.
