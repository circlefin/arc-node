  ## Bridges

  ### Arc Relay Bridge

  A cross-chain USDC bridge built on Circle's CCTP V2.

  - **Chains:** Arc Testnet ↔ Ethereum Sepolia, Base Sepolia, Avalanche Fuji
  - **Protocol:** CCTP V2 burn-and-mint (no wrapped tokens, no liquidity pools)
  - **Stack:** React 19 + Vite + ethers.js v6
  - **Docs:** [docs/ecosystem/arc-relay-bridge.md](arc-relay-bridge.md)
